# Gralobe

Interactive 3D globe visualization that morphs between flat Mercator map and sphere.

## Structure

```
passionforefficiency-submission/   # Main project (TypeScript + Three.js + Vite)
  └── video_demo_1.mov             # Early submission video for mentor review
older_attempts/                    # Earlier HTML prototypes
```

## Run

```bash
cd passionforefficiency-submission
npm install
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
```

Output in `dist/`
